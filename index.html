Representing data in a graphical format can help make information more digestable.

Large and complex amounts of data can be simplified, allowing users to identify trends or relationships.

Both the structure of the data and what we want to learn from it inform our choice of visualization.


For example the development of a stock index like NASDAQ over time is well represented in a line-graph. Interactive elements can provide options to show not only the price but related information like trading volume.


Currently Showing: Open Price



In other cases we may want to compare different populations.

Boxplots give a quick overview over the distribution of a single variable.

Filter options can add some granularity to allow the user to define the exact population of interest.

Often we are not just interested in distributions, but rather the relation between variables.

With two variables we can add our data to a scatterplot and visualize the relation with a regression line.

Adding a selection option for the independent variable allows us to observe multiple relations.

For example, if ratings for wine quality are related to specific chemical compounds.
